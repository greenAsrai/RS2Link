\section{Reaction Systems}

\begin{definition}
A reaction system, abbreviated $rs$, is an ordered pair ${\cal A} = (S, A)$ such that $S$ is a finite set, and $A$ is a
set of reactions in $S$.\\
The set $S$ is called the background (set) of $A$.\\
The main notion concerning the functioning of a reaction system is defined as follows.
\end{definition}
